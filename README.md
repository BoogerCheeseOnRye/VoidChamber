# 🌌 VOIDSCREAM

**A 3×3 neon scream chamber in ONE single HTML file.**

No server. No accounts. No likes. No replies.  
Just pure temporary neon chaos. Scream whatever you want. The void eats everything after 9 screams anyway.

## Features
- Fully responsive 3×3 grid (1 column on mobile, 2 on tablet, 3 on desktop)
- Type **any name** every time — become whoever you want
- Optional image links (no uploads ever)
- Newest scream always appears first
- When you hit 10 screams, the oldest one dramatically **falls away into the abyss** with a sick animation
- Remembers your last screamer name
- Beautiful empty state when the void is truly empty
- Dark humor footer: *"MAX 9 SCREAMS VISIBLE • THE VOID EATS THE REST • THEY NEVER EXISTED ANYWAY"*
- 100% offline & temporary (everything lives only in your browser)

## How to Use
1. Download `voidscream.html`
2. Double-click it (or drag into any browser)
3. Hit the giant glowing **SCREAM** button
4. Type a name + your message + optional image link
5. Watch the neon grid update instantly

**Pro tip:** Open it on your phone — it’s stupidly satisfying in portrait mode.
